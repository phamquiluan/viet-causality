# Vietnamese Researchers in Causality

A curated, community-maintained list of Vietnamese researchers (heritage, citizenship, or affiliation) working worldwide on **Causality**, **Causal Inference**, **Causal Discovery**, and **Causal AI** — both theoretical and applied.

> **Status:** under construction (v0.0). The table below is auto-generated from [`_data/researchers.yml`](_data/researchers.yml). See [CONTRIBUTING.md](CONTRIBUTING.md) to add yourself or someone you know.

## Scope

| Dimension | Included |
|---|---|
| **Vietnamese-ness** | Vietnamese heritage / citizenship / currently affiliated with a Vietnamese institution / Vietnam-born |
| **Seniority** | Full / Associate / Assistant Professors, Senior Research Scientists, Research Fellows, Lecturers, Postdocs, late-stage PhD students with first-author causality publications |
| **Causality scope** | Theoretical (SCMs, do-calculus, identifiability, causal discovery), Causal ML/AI (representation learning, RL, treatment-effect networks), Applied (health, econ, policy, social science), Adjacent (counterfactual reasoning, causal fairness, causal NLP/CV/IR) |

## How entries are sourced

Each entry is tagged with a `source` field — `manual_seed` (hand-curated), `scrape_*` (from OpenAlex / DBLP / Semantic Scholar / arXiv), or `self_submitted` — and a `confidence` field (`high` / `medium` / `low`). See [methodology](#methodology) below.

## Methodology

This list is built with the following pipeline:

1. **Seed** — hand-curated from known senior Vietnamese researchers in the field.
2. **Database sweep** — OpenAlex (causal-inference concept `C158600405` + country/surname filters), Semantic Scholar Graph API, DBLP venue-restricted queries (UAI, NeurIPS, ICML, ICLR, AISTATS, CLeaR, AAAI, IJCAI, KDD), and arXiv (`stat.ME`, `stat.ML`, `cs.LG`).
3. **Name disambiguation** — a tiered Vietnamese surname allowlist (high-confidence: Nguyen, Tran, Pham, Truong, Ngo, Huynh, Phan, Dang, Bui, Doan, Trinh; lower-confidence: Le, Vu, Vo, Lam, Cao, Ho, Ly, Hoang, Duong, Dinh, etc.) combined with second signals (Vietnamese given-name tokens, `Van`/`Thi` middle-name marker, or affiliation history at a Vietnamese institution).
4. **Snowball** — for each confirmed researcher, co-author graphs from OpenAlex / DBLP are walked to surface additional candidates.
5. **Community verification** — researchers may self-submit, request corrections, or opt out via a GitHub issue.
