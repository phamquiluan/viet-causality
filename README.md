## Methodology

This list is built with the following pipeline:

1. **Seed** — hand-curated from known senior Vietnamese researchers in the field.
2. **Database sweep** — OpenAlex (causal-inference concept `C158600405` + country/surname filters), Semantic Scholar Graph API, DBLP venue-restricted queries (UAI, NeurIPS, ICML, ICLR, AISTATS, CLeaR, AAAI, IJCAI, KDD), and arXiv (`stat.ME`, `stat.ML`, `cs.LG`).
3. **Name disambiguation** — a tiered Vietnamese surname allowlist (high-confidence: Nguyen, Tran, Pham, Truong, Ngo, Huynh, Phan, Dang, Bui, Doan, Trinh; lower-confidence: Le, Vu, Vo, Lam, Cao, Ho, Ly, Hoang, Duong, Dinh, etc.) combined with second signals (Vietnamese given-name tokens, `Van`/`Thi` middle-name marker, or affiliation history at a Vietnamese institution).
4. **Snowball** — for each confirmed researcher, co-author graphs from OpenAlex / DBLP are walked to surface additional candidates.
5. **Community verification** — researchers may self-submit, request corrections, or opt out via a GitHub issue.
